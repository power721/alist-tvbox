## Phase 1: telegram-search索引服务

### Task 1: 基础结构

**文件：**
- `internal/alist115/model.go` - 数据模型
- `internal/alist115/service.go` - 索引服务
- `internal/alist115/path_mapper.go` - 路径处理

**核心功能：**
- 数据模型（IndexNode, SearchRequest）
- bleve索引初始化
- 路径映射（emoji → 普通）

**测试：**
```bash
go test ./internal/alist115 -v
```

---

### Task 2: 批量导入

**文件：**
- `internal/alist115/importer.go` - 批量导入逻辑

**核心功能：**
- BatchIndex(nodes []IndexNode) 批量写入bleve
- 路径映射应用
- 错误处理

---

### Task 3: 搜索功能

**文件：**
- `internal/alist115/searcher.go` - 搜索封装

**核心功能：**
- Search(query, page, perPage) 全文搜索
- 分页支持
- 路径/文件名提取

---

### Task 4: HTTP API

**文件：**
- `internal/api/alist115_handlers.go` - 处理器
- `internal/api/router.go` - 路由注册

**端点：**
- POST `/api/external/115/import-batch`
- GET `/api/external/115/search`
- DELETE `/api/external/115/clear`

**测试：**
```bash
curl -X POST http://localhost:8080/api/external/115/import-batch \
  -H "Content-Type: application/json" \
  -d '{"nodes":[{"path":"/test.mkv","size":123,"indexed_at":"2024-01-01T00:00:00Z"}]}'
  
curl "http://localhost:8080/api/external/115/search?q=test"
```
