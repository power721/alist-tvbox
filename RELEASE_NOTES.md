# Release Notes - 1.13.0

## 新增

- 新增 115 分享索引：当配置了 115 账号时，高级设置显示一键下载入口，可下载并更新本地索引
- 115 索引内置为站点，支持浏览、搜索与播放
- 新增「登录会话」管理：用户页可查看当前账号的全部登录会话，支持按会话撤销登录
- 登录会话自动记录登录 IP 与客户端浏览器、操作系统信息
- 首次启动时向 MySQL、PostgreSQL 注入基础豆瓣电影数据，补齐非 H2 环境的默认数据
- 115分享索引：在线更新热加载

## 优化

- 优化电报频道搜索体验

## 修复

- 修复 MySQL、PostgreSQL 环境下豆瓣电影数据相关的问题
- 115分享索引：修复任务
- 115分享索引：修复搜索播放
- 支持新版索引id
- 修复数据库枚举值
