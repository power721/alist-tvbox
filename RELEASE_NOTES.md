# Release Notes - v1.10.0

## 新增

- 数据库导出与导入：支持 JSON、YAML 两种格式，可在 Web 端导出 / 导入数据库；备份与还原同时兼容 SQL 与 JSON，自动识别备份格式并以 JSON 优先
- 新增受 token 保护的 `/api/local/backup` 备份接口（JSON 为主，SQL 为回退）
- 部署脚本内置帮助命令：支持 `help` / `-h` / `--help` 快速查看可用命令
- 所有备份文件名统一加时间戳（`yyyy-MM-dd-HHmmss`），便于区分多次备份

## 优化

- 备份数据库时排除 ALIAS，减少无关数据，缩小备份体积

## 修复

- 修复索引下载失败的问题
- 修复 Web 端导出时下载文件名缺少时间戳的问题
